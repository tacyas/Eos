mrcImageSymmetryCentreFind

Usage

Usage: mrcImageSymmetryCentreFind
Options:
    [-i[nput]            In                  (NULL      ).as(inFile              ) ] :Essential :Input: mrcImage
    [-o[utput]           Out                 (NULL      ).as(outFile             ) ] :Optional  :Output: mrcImage
    [-O[utput]           Out2                (stdout    ).as(outFile             ) ] :Optional  :Output: ASCII
    [-c[onfig]           configFile          (NULL      ).as(inFile              ) ] :Optional  :ConfigurationFile
    [-m[ode]             mode                (0         ).as(Integer             ) ] :Optional  :Mode
----- Additional Usage -----
>>>> check type mode
    1 : Check by Add 
    2 : Check by Correlation 
    4 : Check by Fourier
    8 : Check by TTest
    16 : Check by SRP
>>>> centre type mode
    0 : Centre is GC (gravity of centre)
    1 : Centre is Rotational Symmetry
    2 : Centre is at Image Centre 
    3 : Centre is at Image Origin
>>>> subtrcation type mode
    0 : No Subtraction
    1 : Mean Subtraction
    2 : Min Subtraction
