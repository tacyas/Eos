mrcImageConnectivityNumberCalc

Usage

Usage: mrcImageConnectivityNumberCalc
Options:
    [-i[nput]            In                  (NULL      ).as(inFile              ) ] :Essential :InputDataFile
    [-o[utput]           Out                 (NULL      ).as(outFile             ) ] :Essential :OutputDataFile
    [-c[onfig]           configFile          (NULL      ).as(inFile              ) ] :Optional  :ConfigurationFile
    [-m[ode]             mode                (0         ).as(Integer             ) ] :Optional  :Mode
    [-f[orm]             form                (0         ).as(Integer             ) ] :Optional  :Form
    [-I[nvers]           invers              (0         ).as(Integer             ) ] :Optional  :Invers
----- Additional Usage -----
mode   0 : Four Neighborhood. [default] : For 2 Dimension. 
       1 : Eight Nieghborhood.          : 
       2 : Eighteen Neighborhood.       : For 3 Dimension. 

form   0 : No change. [default]      
       1 : Spread of pixel data 1.   
       2 : Thin of pixel data 1.     
       3 : Frame of image by data 1. 

invers 0 : The object is 1 value. [default] 
       1 : The background is 1 value.
