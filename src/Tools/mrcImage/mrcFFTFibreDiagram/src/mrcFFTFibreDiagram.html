mrcFFTFibreDiagram

Usage

Usage: mrcFFTFibreDiagram
Options:
    [-i[nput]            In                  (NULL      ).as(inFile              ) ] :Essential :Input: HelicalObject/FFT3D or Image3D
    [-o[utput]           Out                 (NULL      ).as(outFile             ) ] :Essential :Output: 2D mrcImage

    [-m[ode]             mode                (0         ).as(Integer             ) ] :Optional  :Mode
----- Additional Usage -----
