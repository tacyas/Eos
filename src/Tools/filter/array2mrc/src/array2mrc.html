array2mrc

Usage

Usage: array2mrc
Options:
    [-i[nput]            In                  (NULL      ).as(inFile              ) ] :Essential :InputDataFile
    [-o[utput]           Out                 (NULL      ).as(outFile             ) ] :Essential :OutputDataFile
    [-c[onfig]           configFile          (NULL      ).as(inFile              ) ] :Optional  :ConfigurationFile
    [-a[xisOrder]        ax                  (1         ).as(Integer             ) 
                         ay                  (2         ).as(Integer             ) 
                         az                  (3         ).as(Integer             ) ] :Optional  :AxisOrder
    [-m[ode]             mode                (0         ).as(Integer             ) ] :Optional  :Mode
    [-m[ode]m[rcimage]   mm                  (2         ).as(Integer             ) ] :Optional  :Mode of mrcImage
    [-l[ength]m[rcimage] lm                  (0         ).as(Real                ) ] :Optional  :Length of mrcImage
    [-l[ength]m[rcimage]3lmx                 (0         ).as(Real                ) 
                         lmy                 (0         ).as(Real                ) 
                         lmz                 (0         ).as(Real                ) ] :Optional  :Length of mrcImage
