----- Additional Usage -----
----- -Mode -----
0: Normal Correlation FxG*
1: Phase  Correlation FxG*/     |FxG*|

3: Normalized Normal Correlation FxG*/|F||G|)
16: (0,0) = 0 
Refinement Correlation Map
0: sqrt(Sum |Fi-Gi|^2                 / Sum |Gi|^2)
