mrcImageDistanceConversion

距離関数変換


オプション一覧
オプション 必須項目/選択項目 NULL デフォルト
-i 必須 入力ファイル設定 NULL
-o 選択 出力ファイル設定 NULL
-m 選択 モード 0
-f 選択 フォーム 0
-h 選択 ヘルプを表示 　


モードの詳細
モード 説明
0 ４近傍
1 ８近傍


フォームの詳細
フォーム 説明
0 距離転換のみ
1 距離転換＋マスキングフレーム
