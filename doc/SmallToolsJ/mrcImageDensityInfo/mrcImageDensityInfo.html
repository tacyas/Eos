mrcImageDensityInfo

密度の詳細を表示


オプション一覧
オプション 必須項目/選択項目 NULL デフォルト
-i 必須 入力ファイル設定 NULL
-o 選択 出力ファイル設定 NULL
-C 選択 データの刻み幅を設定 10
-log 選択 NULL
-m 選択 モード 0
-h 選択 ヘルプを表示 　


モードの詳細

1 体積百分率
2 重心
4 最大値
8 重心の周りの標準偏差
16 最大値の周りの標準偏差
