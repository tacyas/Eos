mrcImageSubtraction

画像を減色させる


オプション一覧
オプション 必須項目/選択項目 NULL デフォルト
-i1 必須 入力ファイル設定 NULL
-i2 必須 入力ファイル設定 NULL
-o 必須 出力ファイル設定 NULL
-m 選択 モード 0
-h 選択 ヘルプを表示 　


モードの詳細
モード 説明
0 out = in1 - in2
（出力結果＝入力ファイル１−入力ファイル２）
1 out = abs (in1 - in2)
（出力結果＝｜（入力ファイル１−入力ファイル２）｜）
