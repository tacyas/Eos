mrcImagePixelDataGet

ピクセルのデータを表示


オプション一覧
オプション 必須項目/選択項目 説明 デフォルト
-i 必須 入力ファイル設定 NULL
-o 選択 出力ファイル設定 NULL
-x 選択 xの値設定 0.0
-y 選択 yの値設定 0.0
-z 選択 zの値設定 0.0
-h 選択 ヘルプを表示
