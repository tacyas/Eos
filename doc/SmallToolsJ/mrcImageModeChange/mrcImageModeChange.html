mrcImageModeChange

画像の状態の変更


オプション一覧
オプション 必須項目/選択項目 NULL デフォルト
-i 必須 入力ファイル設定 NULL
-o 必須 出力ファイル設定 NULL
-Min 選択 最小値を設定 0.0
-Max 選択 最大値を設定 1.0
-m 選択 モード 　
-h 選択 ヘルプを表示 　

モードの詳細
モード 説明
0 １バイト（cher）
2 ４バイト（float）
