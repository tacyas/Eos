mrcImageROI3D

3D画像での関心領域設定


オプション一覧
オプション 必須項目/選択項目 NULL デフォルト
-i 必須 入力ファイル設定 NULL
-o 必須 出力ファイル設定 NULL
-x 選択 x1からx2までを選択範囲とする x1=0.0 x2=0.0
-y 選択 y1からy2までを選択範囲とする y1=0.0 y2=0.0
-z 選択 z1からz2までを選択範囲とする z1=0.0 z2=0.0
-h 選択 ヘルプを表示
