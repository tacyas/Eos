mrcImageMaskingByImage


オプション一覧
オプション 必須項目/選択項目 NULL デフォルト
-i 必須 入力ファイル設定 NULL
-o 必須 出力ファイル設定 NULL
-M 必須 出力ファイル設定 NULL
-v 選択 マスク処理の閾値を設定する 0
-m 選択 モード 0
-h 選択 ヘルプを表示 　


モードの詳細
モード 説明
0 オプション-vの値より大きい値の部分をマスク処理する
1 オプション-vの値より小さい値の部分をマスク処理する
2 マルチファイルと入力ファイルを掛け合わせる
+16 密度から対象物の平均値を引く
