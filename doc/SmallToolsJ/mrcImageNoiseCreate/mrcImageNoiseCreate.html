mrcImageNoiseCreate

ノイズを作る


オプション一覧
オプション 必須項目/選択項目 説明 デフォルト
-r 必須 参照ファイル設定 NULL
-o 必須 出力ファイル設定 NULL
-s 選択 ノイズの標準偏差値を設定 1.0
-h 選択 ヘルプを表示 　

※出力ファイルは、参照ファイルと同じ情報（Nx,Ny,Nz,Lengthなど）を持つ
