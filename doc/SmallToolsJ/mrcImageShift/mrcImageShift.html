mrcImageShift

mrcImage画像をシフトさせる


オプション一覧
オプション 必須項目/選択項目 説明 デフォルト
-i 必須 入力ファイル設定 NULL
-o 必須 出力ファイル設定 NULL
-x 選択 x軸の値設定 0.0
-y 選択 y軸の値設定 0.0
-z 選択 z軸の値設定 0.0
-h 選択 ヘルプを表示
