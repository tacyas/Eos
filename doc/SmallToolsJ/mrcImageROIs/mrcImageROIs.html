mrcImageROI

関心領域設定（複数）


オプション一覧
オプション 必須項目/選択項目 説明 デフォルト
-i 必須 入力ファイル設定 NULL
-I 必須 Infomationファイルの設定 NULL
-o 選択 合成ファイル設定 NULL
-S 選択 拡大・収縮させる NULL
-width 選択 幅を設定 0.0
-height 選択 高さを設定 0.0
-M 選択 モード１ 2
-m 選択 モード２ 0
-h 選択 ヘルプを表示 　

モード１の詳細
モード 説明
0 byte
1 short
2 float

モード２の詳細
モード 説明
0 最短距離法
1 線形補間法
2 ３次補間法
3 多項式法
