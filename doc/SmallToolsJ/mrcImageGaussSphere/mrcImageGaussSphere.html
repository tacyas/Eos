mrcImageGaussSphere


オプション一覧
オプション 必須項目/選択項目 NULL デフォルト
-o 必須 出力ファイル設定 NULL
-sigmax 選択 物体のｘ軸の幅を設定 10
-sigmay 選択 物体のｙ軸の幅を設定 20
-sigmaz 選択 物体のz軸の幅を設定 30
-sigma_range 選択 画像データの大きさを設定 3
-weight 選択 物体の高さを設定 10
-size 選択 ピクセルのサイズを設定 5
-W 選択 画像データの幅（ｘ軸）を設定 0
-H 選択 画像データの高さ（ｙ軸）を設定 0
-S 選択 画像データの奥行き（ｚ軸）を設定 0
-m 選択 モード 2
-h 選択 ヘルプを表示 　
ファイルサイズ＝(sigma_range*sigmax*2) x (sigma_range*sigmay*2) x (sigma_range*sigmaz*2)


モードの詳細
モード 説明
0 char（１バイト）
2 float（４バイト）
